Questo layout di tastiera per Windows è stato creato con
[MS keyboard layout creator 1.4](http://www.microsoft.com/en-us/download/details.aspx?id=22339).

# Installazione

 - Eseguire come amministratore `setup.exe` (in `it-dev`)
 - Aprire il pannello di controllo di Windows, poi *Orologio* > *Lingua*. Aprire il menu
*Impostazioni avanzate* sulla sinistra, scorrere fino alla sezione sul metodo di input predefinito
e scegliere il layout appena installato
 - Logout/login
